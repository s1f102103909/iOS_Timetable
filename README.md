# 課題管理 × 時間割アプリ（iOS）

## 概要
GeekSalonのiPhoneコースで開発した、**課題管理**と**時間割**を一体化したiOSアプリです。  
「**とにかく便利で、他と被らない作品**」をコンセプトに、課題の提出忘れを防ぐための通知設計にこだわりました。

- **使用言語**: Swift
- **開発**: GeekSalon iPhoneコース（アプリ開発）

---

## ターゲット
- 課題の提出忘れを防ぎたい **高校生 / 大学生**
- 締切が近づくまで課題の存在を忘れがちな人

---

## 制作背景（なぜ作ったか）
課題の提出忘れを防ぐためです。  
「課題を登録していないと通知できない」という弱点を補うため、時間割側からも提出忘れを防げる仕組みを作りました。

---

## 主な機能
### 1. 課題管理（リマインド通知）
課題を登録すると、提出期限前に通知でリマインドします。

- 提出 **1時間前**
- 提出 **1日前**
- 提出 **2日前**

### 2. 時間割（授業登録）
時間割に授業を登録できます。

### 3. “課題をやったか？” を促す通知（工夫ポイント）
時間割に登録した授業に対して、**授業日の2日前**に  
「その授業の課題は終わった？」という通知を送ります。

> 課題管理に課題を登録しない人もいると考え、  
> **時間割に授業だけ登録していても**提出忘れを防げるように設計しました。
